# Data Collection

Three publicly available customer churn datasets were collected for exploration
and evaluation as part of the capstone project.

## Dataset 1: SaaS Subscription & Churn Analytics Dataset
- Source: Kaggle
- Description: Simulated SaaS customer behavior data including subscription
  activity, usage patterns, and churn labels.
- Collection Method: Downloaded directly from Kaggle in CSV format.

## Dataset 2: Customer Churn Prediction Business Dataset
- Source: Kaggle
- Description: Customer demographic, subscription, and churn information designed
  for supervised machine learning classification tasks.
- Collection Method: Downloaded directly from Kaggle in CSV format.

## Dataset 3: Ecommerce Churn Dataset
- Source: Kaggle
- Description: Transactional and behavioral customer data used to analyze churn
  in an ecommerce context.
- Collection Method: Downloaded directly from Kaggle in CSV format.

Due to file size constraints, only small sample subsets of each dataset are stored
in this repository. Full datasets are available via their original sources.
